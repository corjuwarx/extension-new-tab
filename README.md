# New Tab Extension

A modern, fast, and customizable browser new tab extension.

## Features
- **Quick Search**: Support for Brave, Google, DuckDuckGo, Bing, and Yandex.
- **Quick Links**: Easy access to your most-used sites.
- **Extensive Icon Library**: Over 40 ready-to-use icons.
- **Custom Icon Upload**: Ability to use your own images as icons.
- **Modern UI**: Dark mode compatible, Glassmorphism design.
- **Privacy Focused**: Your data is stored locally.

## Installation
1. Download or clone this repository.
2. Open the `extensions` page in your browser (Brave, Chrome, Edge, etc.).
3. Enable "Developer Mode".
4. Click "Load unpacked" and select this project folder.
